Proyecto del reto **#30-Días-de-JavaScript**, inspirado en el curso [**JavaScript30** de Wes Bos](https://github.com/wesbos/Wes-Bos-Video-Notes/tree/master).
# Día 1 - Tambores | #30-Días-de-JS
## Descripción
Este mini proyecto reproduce sonidos de batería al presionar teclas específicas del teclado.  
Cada tecla está asociada a un sonido diferente, logrando una experiencia interactiva y divertida con JavaScript puro.

# Día 2 - Reloj | #30-Días-de-JS
## Descripción
En este proyecto, se genera un reloj digital que muestra la hora actual en formato HH:MM:SS.

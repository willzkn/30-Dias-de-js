# Día 1 - Tambores | #30-Días-de-JS

Proyecto del **Día 1** del reto **#30-Días-de-JavaScript**, inspirado en el curso [**JavaScript30** de Wes Bos](https://github.com/wesbos/Wes-Bos-Video-Notes/tree/master).

## Descripción
Este mini proyecto reproduce sonidos de batería al presionar teclas específicas del teclado.  
Cada tecla está asociada a un sonido diferente, logrando una experiencia interactiva y divertida con JavaScript puro.
